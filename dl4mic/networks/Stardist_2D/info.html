Segment Nuclei using StarDist

StarDist is a deep-learning method that can be used to segment cell nuclei in 2D (XY) single images or in stacks (XYT), first published by Uwe Schmidt et al.. StarDist predicts object probabilities and star-convex polygons parameterized by the radial distances using a simple U-Net architecture and by selecting the final instances via non-maximum suppression.
